# ParkPilot

**ParkPilot** is a website to reserve a place in a car parking garage in a mall.  

## Features
- View available parking spots.
- Reserve a spot ahead of time.
- User-friendly front-end layout with HTML and CSS.


## Pages
1. **Home Page:** Introduction and overview of the ParkPilot service.
2. **Ready to Park Page:** Explains how to use the app to park.
3. **Form Page:** Allows users to book a parking spot by entering their details.

## Technologies Used
- HTML
- CSS
- Basic responsive design techniques

## How to Use
1. Open the `index.html` file in a web browser.
2. Navigate through the pages to explore the features.
3. Fill out the form to simulate reserving a parking spot.
